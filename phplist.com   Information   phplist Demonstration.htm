phplist Demonstration

Everything works in the demo, except the actual delivery of email messages.

  • Admin URL: demo.phplist.com/lists/admin/
  • Admin login: admin
  • Admin password: phplist
  • Frontend URL: demo.phplist.com/lists/
Please note: the demo DB is restored every so often. If you are playing around with it, your changes may be wiped while working on it. Also, if the above password does not work it will start working again a little later.

  • Admin URL: dev.phplist.com/lists/admin/
